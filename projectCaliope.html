Caliope – Literary Social Network Platform

Project Overview

Caliope is a full-stack social networking platform developed during a two-month Java Full Stack Bootcamp capstone project. The platform allows users to publish, discover, review, and rate literary works, creating an interactive community for readers and writers.

The project was designed to combine social networking features with literary content sharing, encouraging user engagement through reviews, ratings, and discussions..

Role

Product Owner | Scrum Master | Full-Stack Developer

Throughout the project, I assumed multiple responsibilities, contributing both to project management and software development. This experience allowed me to participate in every stage of the software development lifecycle, from planning and requirements gathering to implementation and team coordination.

Technologies Used

  • Java
  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap
  • Git & GitHub
  • Scrum Methodology

Key Contributions

  • Defined and prioritized user stories to align development efforts with project goals.
  • Facilitated Scrum ceremonies and coordinated team activities as Scrum Master.
  • Developed both front-end and back-end functionalities.
  • Created responsive and user-friendly interfaces to improve user experience.
  • Participated in the implementation of features that enable users to publish, browse, and rate literary works.
  • Contributed to system integration and collaboration across multiple development areas.

Challenges and Solutions

Managing multiple roles simultaneously represented a significant challenge. Balancing leadership responsibilities with technical development required strong organization, effective communication, and continuous prioritization. By applying Scrum practices and maintaining close collaboration with the team, we successfully delivered a functional and scalable platform within the project timeline.
